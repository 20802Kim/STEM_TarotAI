# TarotAI

made by brAIns

## How to use:

1. download all the files.

```shell
git clone https://github.com/dirac042/TarotAI.git
```

2. install all the requirements.

   1. using makefile

   ```shell
   make install
   ```

   2. using install.sh

   ```shell
   chmod +x install.sh
   ./install.sh
   ```

3. run 'main.py' via Shell.
